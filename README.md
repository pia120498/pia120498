# 💫 About Me:
Hello World! I'm Priya, a DevOps Engineer from India.<br><br>Passionate about Cloud, DevOps, and Infrastructure Automation with experience in Linux, AWS, SQL, and Database Administration. Currently transitioning into a dedicated DevOps role by building hands-on projects using Terraform, Ansible, Docker, Kubernetes, and CI/CD pipelines. I enjoy automating repetitive tasks, managing cloud infrastructure, and learning modern DevOps practices. My focus is on Infrastructure as Code (IaC), Configuration Management, Containerization, and Cloud-Native technologies.
